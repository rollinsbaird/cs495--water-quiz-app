Water Quiz Project

This web application is designed to provide quizzes on the subject of water for both education and entertainment purposes.

Client: Dr. Steven Burian, Dr. James Halgren, and the National Water Institute.

Team Members: Rollins Baird, Sam Hertzler, Thomas Hampton

Sprint 1 Goals:

  1. Find a Javascript framework to use.
  2. Implement backend system to use for our PWA and quiz data.
  3. Create a basic draft for homepage & quiz UI.
  4. Configure GitHub, Jira, & additional tools.
  5. Make sure everyone has the same tools and on the same page with goals.

How it Works:
  • A user will navigate to the web app on either a mobile device or laptop
  • The homescreen consists of a credits page with a dropdown menu of quizzes available with tags to search through and a link to add a quiz
    • The link to add a quiz will take the user through an interactive process of addidng and viewing a new quiz
  • The user can click on a a quiz based on certain tags such as difficulty or location
    • Each quiz will range in question count and type
    • With the competion of each quiz a score will be displayed as well as a chance to retake the quiz
    • The user will also see a leaderboad for last 30 minutes, as well as all time
      • Each leaderboard can also be found from a dropdown menu on the home screen
