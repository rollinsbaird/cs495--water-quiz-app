HydroGenius

This web application is designed to provide quizzes on the subject of water for both education and entertainment purposes.

    Client: Dr. Steven Burian, Dr. James Halgren, and the National Water Institute.

    Team Members: Rollins Baird, Sam Hertzler, Thomas Hampton

    Goals:

    1. Implement backend system to use for our PWA and quiz data.
    2. Create a homepage, quiz UI, quiz selection, and leaderboard page for quizzes.
    3. Create robust sample content.
    4. Add a logo and share buttons.
    5. Documentation for handoff.

    Approach:

    How it Works:
    • A user will navigate to the web app on either a mobile device or laptop
    • The homescreen consists of a credits page with a button that links to the quiz selection pages
    • The quiz selection page contains a list of quizzes displayed as cards
      • Each quiz has a title and a difficulty tag as well as a short description under the title
    • There are two options for each quiz card, the user can click the card or the user can click the a trophy emoji in the corner
      • Clicking the card takes the user to a specific quiz
      • Clicking the trophy emoji takes the user to a specific leaderboad page
    • Each quiz consists of multiple choice questions that have optional informartion screen before the question and an optional feedback screen after the question has been answered
      • At the conclusion of each quiz the user will see a closiong card
        • A personalized score
        • An option to input the user's name to be saved to the leaderboard
        • A share button to share the score and a short snippet of about the application to social media
        • A button to return to the quiz selection page and a button to restart the quiz
    • The leaderboard page consists of two buttons
      • One button dislays all time scores, the other displays scores from the last hour
      • The scores are sorted by percentage
